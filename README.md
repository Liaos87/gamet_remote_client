# 一，说明
此为格姆特远程控制客户端的flutter版本，项目的配置文件中已存在android，ios的配置文件，
主要实现功能为链接特定ip等参数·，发送http等请求到服务端完成控制播放暂停停止等视频相关
功能，后续将继续开发对应功能。
